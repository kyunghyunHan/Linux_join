# 개인프로젝트 (마리아디비+우분투+로그인,회원가입+게시판)

## 설명
- sns 구현
- 도커 우분투 서버에서의 개인 프로젝트 작성
- 마리아 디비연동 (rds+마리아디비)
- 로그인및 회원가입 구현
- 이미지 업로드 및 게시판 구현(SNS)
